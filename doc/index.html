Overview

Kyoto Tycoon is a lightweight database server with auto expiration mechanism, which is useful to handle cache data of various applications.

Tutorial

Bacause Kyoto Tycoon uses Kyoto Cabinet, install KC beforehand. Then, install KT.

$ tar zxvf kyototycoon-x.y.z.tar.gz
$ cd kyototycoon-x.y.z.tar.gz
$ ./configure
$ make
$ make check
$ sudo make install

Run the server. To stop it, input Ctrl-C on the terminal.

$ ktserver

Access the server from another terminal.

$ ktremotemgr set foo bar         # store a record
$ ktremotemgr get foo             # retrieve it
$ ktremotemgr set -xt 10 foo bar  # store again with the expiration time

# wait for 10 seconds or more

$ ktremotemgr get foo             # the record will be expired
